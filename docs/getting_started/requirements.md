# Requirements

## Software

- Python: 3.7 to 3.11

## External Dependencies For Shift Interactive

- QtPy: 1.9
- PySide2: 5.15

## System Requirements

- **OS**: Microsoft *Windows 10* or higher, *Linux CentOS 7* and *Rocky Linux 9*
- **CPU**: 64-bit Intel or AMD multi-core processor
- **RAM**: 256MB suggested
- **Storage**: at least 500MB of free disk space.
